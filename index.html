What is the purpose of these tags?

<img>

Embeds images into an HTML document using the src attribute, the value of which is the address of the embedded image.

<textarea>

Creates large text input fields.

<title>

The title of the HTML document displayed at the top of the browser title bar. It can also be displayed in search results, so it should be taken into account the provision of the name.

<a>

Creates hypertext links.

<b>

Sets the bold outline of a text passage without giving emphasis or importance to the highlighted one.

<body>

Represents the document body (content that is not related to the document metadata).

<br>

Text transfer to a new line.

<div>

A container element for sections of an HTML document. It is used for grouping block elements for the purpose of formatting styles.

<li>

An element of a bulleted or numbered list.
